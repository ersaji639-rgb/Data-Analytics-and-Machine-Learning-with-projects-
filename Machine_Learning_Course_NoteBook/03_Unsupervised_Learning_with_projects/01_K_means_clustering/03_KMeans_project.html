This is unsupervised learning, so there are no labels. We train the model using the data to find cluster centroids. After training, the model groups the data into three clusters. We then interpret the meaning of each cluster based on the features of the data points in that cluster. When new data arrives, we use the same trained model to predict which cluster it belongs to. Based on the previous interpretation of that cluster, we can infer the likely characteristics of the new data.
